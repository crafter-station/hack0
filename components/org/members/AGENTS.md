# Members Module

## Purpose
Display público de miembros y gestión administrativa.

## Components
| Component | Access | Description |
|-----------|--------|-------------|
| MemberShowcaseGrid | Public | Galería de badges |
| MemberShowcaseCard | Public | Card individual |
| MembersManagement | Admin | Tabla de gestión |
| InviteManager | Admin | Gestión de invites |
| AcceptInviteButton | Public | Aceptar invitación |
| JoinOrgDialog | Public | Dialog de unirse |
| FollowButtonAnimated | Public | Botón follow animado |

## Roles
owner > admin > member > follower
