# Settings Module

## Purpose
Configuración de organización. Todos los componentes son Admin-only.

## Components
- SettingsSidebar - Navegación de settings
- OrgSettingsForm - Form principal
- LocationSelector - Picker de país/región

## Navigation
/settings → General
/settings/members → Gestión de miembros (invites, roles, eliminar)
/settings/badge → Badge config (usa badges/)
/settings/campaigns → Campañas (usa campaigns/)

## Note
La galería de badges/insignias ahora está en /comunidad (tab pública)
